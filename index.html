Download the JSON translation files from the Hytale Community Translation Project on Crowdin, then upload them here.

Choose the target language you want to convert to, and click the Convert button to begin.

The conversion will generate .lang files. These files must be placed in the game's language directory.

If you installed the game in the default location, press Win + R, then paste the following path into the dialog and press Enter: %AppData%\Hytale\install\release\package\game\latest\Client\Data\Shared\Language (click to copy)

Inside the Language directory, place the generated file into the appropriate language folder. If the folder does not exist, create one using the language code as the folder name (for example: fr-FR).

Once complete, launch the game and select the new language from the language selection menu.
